# Run and deploy your AI Studio app

This contains everything you need to run your app locally.
https://ai.studio/apps/0d217b45-3866-4578-90f3-6b4ab6f051e9

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

This is a static Vite frontend. Do not configure a Node/Express entry file such as `server.js`.

- Build command: `npm run build`
- Output directory: `dist`
- Entry file: leave blank / not applicable
